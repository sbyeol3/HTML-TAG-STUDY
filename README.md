## 스터디 규칙
- 주차별로 분석할 HTML 태그를 2개 정한다. (일요일 저녁 8시)
- 각 스터디 원은 다음 스터디 전날까지 태그 공부한 내용을 자신의 remote 저장소에 push한 후 master에 PR을 날린다. (master <- fork master)
- 태그 공부한 내용에는 이론 + 각 태그를 응용한 간단한 HTML 코드도 포함한다.
